Pause On Zone Load
- A simple mod that will pause the game when you load into a zone, allowing you to take in the scenery before moving your Sims, or whatever it is you crazy kids do.

Requirements:
- This mod requires at least version 1.75 of Sims 4 Community Library.

Installation:
- Install the Requirements listed above.
- Drop the script file into your Mods folder.
- Enjoy
